[schemabenchmarks.dev](https://schemabenchmarks.dev) is an open source project aiming to provide detailed and transparent comparisons of schema validation libraries.

It measures many aspects of schema libraries, including:

- Bundle size (and thus download time)
- Initialisation time
- Validation time
- Parse time

Visually, it's heavily inspired by my personal site (which you're currently on), though it uses [React](https://reactjs.org) and [Tanstack Start](https://tanstack.com/start/latest), whereas this site uses [Lit](https://lit.dev) and [Greenwood](https://greenwoodjs.dev).
